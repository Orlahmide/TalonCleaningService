# Talon Cleaning Services - React + Vite + Tailwind Scaffold

## Quick start (after downloading)

1. Install dependencies
```
npm install
```

2. Start dev server
```
npm run dev
```

3. Open the site at the address shown in your terminal (usually http://localhost:5173)

## Notes / What you must change

- The contact form posts to Formspree. Replace the action URL in `src/pages/Contact.jsx` with your Formspree endpoint (see https://formspree.io/) or another backend endpoint that sends emails.
- Update content (About, Services, Founder) in `src/pages/*` using the content from your document.
- Add real images in `src/assets` and replace the hero placeholder.
- Update brand colors in `src/index.css` (CSS variables at the top).

## Build for production
```
npm run build
```
